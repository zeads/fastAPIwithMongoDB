## setting mongodb
# jalankan docker client, kemudian di terminal ketik : 
    -> docker run -d -p 27017:27017 --name my_mongodb mongo
    -> docker logs my_mongodb (u/ cek db yang baru dibuat)
    -> docker ps (u/ cek image docker yg berjalan)

    -> docker images (u/ melihat list image docker)
    -> docker start <nama_image> (u/ menjalankan docker image yang sudah ada) atau
    -> docker run -d <nama_image> (u/ membuat kontainer baru lalu menjalankannya) 

# menjalankan aplikasi fastAPI
    -> uvicorn <nama_file_main>:<nama_instant_fastAPI> --reload
        contoh :    uvicorn app:app --reload (jika nama file app)
                    uvicorn main:app --reload (jika nama file main)
